## AI Systems and Applied Generative AI

Recently I’ve been spending a lot of time working with **retrieval-augmented generation (RAG), transformers, and AI-driven systems**.

One project I built explores how generative models can interact with structured financial data and historical market information. The system combines:

- **Retrieval-Augmented Generation (RAG)** for grounding responses in curated data
- **Transformer-based language models** for contextual reasoning
- **Pairs trading strategy analysis** for financial insights
- **Backend APIs** that orchestrate prompts, retrieval pipelines, and response generation

The idea was to explore how LLMs can be combined with structured datasets to create systems that **explain complex financial strategies, provide contextual insights, and assist with research workflows**.

This project was developed as part of **CSYE 7380 – Theory and Practical Applications of Generative AI**, where the goal was to experiment with real-world applications of large language models beyond simple chat interfaces.

## Building Full-Stack AI Products

Another area I am deeply focused on is building **AI-powered full-stack platforms**.

One of the projects I’m currently developing is **FitGenie**, an intelligent fitness assistant that generates personalized workout and nutrition plans using AI.

The system architecture includes:

- **React + Tailwind** frontend
- **FastAPI backend**
- **PostgreSQL + SQLAlchemy** for data modeling
- **AI plan generation pipelines** for workout and nutrition planning
- **User profile driven recommendations**

The goal of the platform is to move beyond static plans and build a system that can **adapt recommendations based on user goals, activity patterns, and feedback loops**.
